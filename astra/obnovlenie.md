# 

:::lab:true Astra 1.7.0

## Добавлено (feat)

-  Адаптивный битрейт (АБР). Битрейт подстраивается в указанном диапазоне в зависимости от пропускной способности сети

-  Возможность выгрузки логов для отладки(пока не используем)

-  Проверка доступности сервера

-  Сообщение при отсутствии подключения

-  Возможность удаления лицензии

## Критические изменения (crit)

-  [comment:G7K3v]Переработана[/comment] привязка устройств

-  [comment:Y6r8l]Добавлена криптография[/comment] для важных настроек, удалены лишние данные

## Улучшено (impr)

-  Производительность USB камер. Теперь можно использовать адаптеры с большим разрешением по сырому кадру(YUYV). Но будьте внимательны, на luckfox серьезные проблемы с питанием, для мощных адаптеров лучше [comment:TRD0k]"приколхозить"[/comment] внешнее питание.

## Исправлено (fix)

-  Исправление проверки доступности сервера

-  Исправление сообщений Mavlink

-  Исправление завершения работы камеры CSI

:::
