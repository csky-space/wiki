**Camera Module SC3336 3MP** -- это модуль камеры, предназначенный для работы с одноплатными компьютерами серии LuckFox Pico. Он использует 3-мегапиксельный CMOS-сенсор для получения изображений и оснащен широкоугольным объективом.

Техническая характеристика.

| Параметр           | Значение                    |
|--------------------|-----------------------------|
| Датчик изображения | SC3336, CMOS, 1/2.8″        |
| Разрешение записи  | 3 МП (2304 × 1296 пикселей) |
| Угол обзора (FOV)  | 98\.3°                      |
| Формат видео       | H.264 или H.265             |
| Битрейт            | Настраивается в ПО/прошивке |
| Выходное видео     | MIPI CSI (через FFC-кабель) |

### **Важная информация о совместимости и применении**

-  **Совместимость**: Модуль создан для работы с платами серии **LuckFox Pico** (например, LuckFox-Pico, LuckFox-Pico-Plus). Эта камера **не совместима** с моделью **LuckFox Pico Zero**.

-  **Комплектация**: Вместе с модулем поставляется гибкий плоский кабель (FFC), необходимый для подключения к основной плате.

-  **Ключевые особенности**: Благодаря **широкому углу обзора** и заявленной **высокой светочувствительности** модуль хорошо подходит для приложений, требующих захвата изображения в условиях слабого освещения
