title: Через Win32 Disk Imager
---

-  Скачать [прошивку](https://ftp.air-link.space/downloads/astra/firmware/astra-raspios-latest.zip)

-  Скачать **Win32 Disk Imager**

-  Вставить карту памяти в компьютер

-  Откройте **Win32 Disk Imager.** Выберите прошивку: значок папки в поле Image File

   ![](./win32-5.png){width=8842px height=6507px}

-  После выбора прошивки нажмите «Write»

   ![](./win32-2.png){width=8842px height=6507px}

-  Дождитесь прошивки

   ![](./win32-3.png){width=8842px height=6507px}

-  После завершения прошивки закройте программу и извлеките носитель

   ![](./win32-4.png){width=8842px height=6507px}



:::note:true Важно

Для прошивки рекомендуется использовать SD накопитель от **30 Мбит (v30)**

:::
