title: Прошивка
---

-  Скачать прошивку по [ссылке](https://ftp.air-link.space/downloads/astra/firmware/)

-  Скачать [SocToolKit](https://ftp.air-link.space/downloads/astra/) и установить [драйвер](https://ftp.air-link.space/downloads/astra/) для Windows

-  Переведите устройство в режим прошивки: Зажмите кнопку  **Boot** и подключите плату к компьютеру с помощью **USB Type-C кабеля** из комплекта

-  Откройте **SocToolKit.exe**

-  В открывшемся окне выберите чип [comment:W7M0Q]**RV1106**[/comment]

-  В левом верхнем углу выберите интерфейс **USB**. В выпадающем списке появится устройство с меткой **"Maskrom..."** -- выберите его

-  Нажмите кнопку **Firmware...** и выберите файл прошивки update.img

-  Нажмите **Upgrade** и дождитесь завершения установки

-  После завершения прошивки **отключите питание по USB**
