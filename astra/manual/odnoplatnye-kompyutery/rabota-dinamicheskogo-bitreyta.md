**Динамический (переменный) битрейт (VBR)** -- режим кодирования видеопотока стандарта, при котором скорость передачи данных адаптивно меняется для поддержания стабильного качества изображения при минимальном среднем размере файла.

**1\.Принцип работы:**

-  **Анализ содержимого**

   -  Кодер анализирует исходный материал, определяя области с различной степенью сложности

   -  Оцениваются: уровень детализации, характер движения, изменения между кадрами

-  **Распределение битового бюджета**

   -  Создается "битовый бюджет" -- общее количество бит, доступное для кодирования

   -  Сложным участкам выделяется больше ресурсов, простым -- меньше

-  **Интеллектуальное кодирование**

   -  Используются различные алгоритмы предсказания для эффективного представления данных

   -  В сложных участках могут применяться более точные, но ресурсоемкие методы сжатия
